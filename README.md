# Awesome-Agentic-AI

In this repo, we categorize and summarize recent works on agentic AI models, as well as agents for security and software engineering.  
Under each work, we list the summary of key techniques and our key takeaways.  

## [Agentic models and techniques](agent.md)

In this part, we discuss general agentic models and techniques. 
It also include some recent reasoning techniques. 

## [Security agents and agentic training](security.md)

In this part, we categorize the agents based on the security problems they tackle, including vulnerability detection, triage, patching, others, and end-to-end frameworks. 
Under each category, we have techniques and benchmarks. 

## [SWE agents and agentic training](swe.md)

In this part, we summarize recent works on SWE agents, where most of them focusing on code generation and issue resolving. 
Based on the agent scaffold, researchers also design methods to train specific models on SWE tasks.
