# Awesome-Agentic-AI

In this repo, we categorize and summarize recent works on agentic AI models, as well as agents for security and software engineering.  
Under each category, we list the summary of key techniques and our key takeaways.  

## [Agentic models and techniques](agent.md)

Here, we discuss general agentic models and training techniques. 

## [Security agents and agentic training](security.md)

This part focuses on security agents. We categorize the agents based on the security problems they tackle, such as vulnerability detection, triage, patching, etc. 
Under each category, we include the corresponding benchmarks and techniques.

## [SWE agents and agentic training](swe.md)

In this part, we summarize recent works on SWE agents, where most of them focusing on code generation and issue resolving. 
Based on the agent scaffold, researchers also design methods to train specific models on SWE tasks. 

## [General LLM reasoning](reasoning.md)

Finally, we provide an incomplete summary of recent works on general LLM reasoning (without involving agents). This direct has been extensively studied, with many techniques on improving RL-based post-training methods and tricks regarding the training process. Here, we mainly focus on the techniques that provide process reward signals, as well as specific training methods for code reasoning.
